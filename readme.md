# AI Career Roadmap Generator

AI Career Roadmap Generator is a Flask-based web application that helps users generate a personalized career roadmap using Google's Gemini AI.

## Features

- User Registration & Login
- Career Details Form
- AI-Based Career Roadmap
- Dashboard
- SQLite Database
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Flask
- SQLite
- Gemini AI

## Project Structure

- templates/
- static/
- app.py
- ai.py
- database.py
- config.py
- requirements.txt

## How to Run

1. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```

2. Run the project:
   ```
   python app.py
   ```

3. Open:
   ```
   http://127.0.0.1:5000
   ```
